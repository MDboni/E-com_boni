## Overview
A modern, serverless E-Commerce platform built for seamless online shopping experiences. This platform is designed using **React.js, Node.js, Express.js, MongoDB, and Zustand**, and integrates payment processing, email notifications, and secure authentication.  

---

## Tech Stack
- **Frontend:** React.js, Zustand, Bootstrap, DaisyUI  
- **Backend:** Node.js, Express.js, RESTful APIs, JWT Authentication  
- **Database:** MongoDB  
- **Payment:** SSLCommerz  
- **Email Notifications:** Nodemailer  

---

## Features

### Secure Authentication
- Developed secure login and session handling using Node.js and JWT.
- Ensures safe user access and protects sensitive data.

### Payment Integration
- Integrated SSLCommerz for real-time payment processing.
- Validates transactions securely and instantly.

### Product & Cart Management
- Dynamic product listing with filtering and sorting.
- Real-time cart updates using React.js and Zustand.

### Automated Notifications
- Configured Nodemailer to send instant order confirmations.
- Users receive email updates for order status and changes.

### Responsive UI
- Modern, mobile-friendly interface.
- Built with Bootstrap and DaisyUI for a seamless experience across devices.
